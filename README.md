# KOHLIBONG — เกาะลิบง.com

Landing page สำหรับเกาะลิบง จังหวัดตรัง

## โครงสร้าง

- `index.html` — หน้าหลัก
- `css/style.css` — สไตล์
- `js/main.js` — carousel, การ์ด, แผนที่, lightbox
- `better-together/` — ฟอนต์ Better Together

## GitHub Pages

1. เปิด [Settings → Pages](https://github.com/goddarkmarketing/KOHLIBONG/settings/pages)
2. **Source:** เลือก **GitHub Actions** (ไม่ใช่ Deploy from a branch)
3. Push ไป `main` แล้ว Action จะ deploy อัตโนมัติ

URL: https://goddarkmarketing.github.io/KOHLIBONG/

## รันบนเครื่อง

วางใน web server (เช่น XAMPP) แล้วเปิด `index.html` ผ่าน localhost
