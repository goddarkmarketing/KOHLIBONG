# KOHLIBONG — เกาะลิบง.com

Landing page สำหรับเกาะลิบง จังหวัดตรัง

## โครงสร้าง

- `index.html` — หน้าหลัก
- `css/style.css` — สไตล์
- `js/main.js` — carousel, การ์ด, แผนที่, lightbox
- `better-together/` — ฟอนต์ Better Together

## GitHub Pages

1. Push ไป branch `main` — Action จะ deploy ไฟล์ไป branch `gh-pages` อัตโนมัติ
2. เปิด [Settings → Pages](https://github.com/goddarkmarketing/KOHLIBONG/settings/pages)
3. **Source:** Deploy from a branch → **gh-pages** / **/ (root)** → Save

URL: https://goddarkmarketing.github.io/KOHLIBONG/

## รันบนเครื่อง

วางใน web server (เช่น XAMPP) แล้วเปิด `index.html` ผ่าน localhost
